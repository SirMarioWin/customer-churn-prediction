# Customer Churn Prediction

## Overview
This project aims to predict customer churn using machine learning techniques. It focuses on analyzing customer behavior and identifying key factors that lead to churn.

## Tools & Technologies
- Python
- Pandas
- Scikit-learn
- Matplotlib / Seaborn

## Workflow
- Data Cleaning
- Exploratory Data Analysis (EDA)
- Feature Engineering
- Model Training
- Model Evaluation

## Dataset
Telco Customer Churn Dataset

## Objective
To help businesses reduce customer churn by identifying at-risk customers.
